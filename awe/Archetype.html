class awe.Archetype

Available on all platforms

Blueprints for fast Entity construction.

This can be constructed by using the Archetype.build(_) macro. Using this, you can build an archetype by calling it with the components you want the Entity to have.

Example

Archetype.build(Position, Velocity);
